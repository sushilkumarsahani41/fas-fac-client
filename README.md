# Cause I Client

## Prerequisites

- [NodeJS](https://nodejs.org/en) v20.11.1
- [Pnpm](https://pnpm.io/installation)

## Installation

```bash
pnpm install
```

### Running the app

```bash
mv .env.example .env
```

```bash
pnpm install
```

```bash
# watch mode
pnpm run dev

# build for production
pnpm run build

# production mode
pnpm run preview
```

### Generating updated Query Hooks

```bash
pnpm run generate
```
